# Vue Studies

A personal collection of small projects built while learning Vue.

This repository is used to practice core concepts such as:

- Reactivity
- Components
- Props
- Computed properties
- Conditional rendering
- Event handling
- Basic project structure with Vite

## Projects

- Password Checker
- Mood Counter
- To-do List *(coming soon)*

## Purpose

The goal of this repository is to explore Vue through small practical exercises, focusing on understanding how Vue simplifies interface logic compared to vanilla JavaScript.

## Stack

- Vue
- Vite
- JavaScript
- CSS
